# 📖 简介

这是一个伪装成普通计算器的魔术道具。 **魔术效果**：无论观众在最后阶段输入什么乱七八糟的数字，最终按下等于号时，结果永远是\*\*“现在的月+日+时间”\*\*（例如：2月17日 12:22 = `2171222`）。

---

## ✅ 核心成功条件（绝对红线）

要想魔术成功，必须满足以下 **3 个硬性条件**，否则会穿帮：

### 1. 数字不能“爆” (The Safety Limit)

这是最重要的一点！

* **原理**：魔术是靠“当前时间数字”减去“屏幕上的数字”来计算差值的。
* **风险**：如果屏幕上的数字 **大于** 当前时间数字，差值就会变成负数，魔术就露馅了。
* **安全操作**：
  * **建议**：前两步输入的数字，尽量控制在 **4位数** 或 **5位数** 以内。
  * **例子**：`1998` + `2023` = `4021`。这个数远远小于时间数字（比如 2171222），非常安全。
  * **禁止**：不要输入 `999999` 这种巨大的数字。

### 2. 流程必须对 (The Sequence)

触发机关的唯一钥匙是操作顺序，错一步都不行：

* ❌ 错误：输入 -> 按加号 -> 输入 -> 按加号（机关会提前触发，导致逻辑混乱）。
* ✅ **正确**：输入 -> 按加号 -> 输入 -> **按等于号**（算出结果） -> **再按加号**（触发机关）。

### 3. 演技要逼真 (The Performance)

* 在“盲打阶段”，当观众输入的位数凑够了之后，继续按数字键是**没有反应**的。
* **关键话术**：这时候你要说：“好了，命运已经注定，多一个数都输不进去。”或者直接引导：“好，现在按下等于号揭晓命运。”

## 🎭 标准表演流程 (Script)

### 第一阶段：铺垫（正常模式）

1. 拿出计算器，随便按两下清零，证明它是正常的。
2. 让观众输入一个对他有意义的数字（比如出生年份，建议4位数）。
   * *观众输入：`1995`*
3. **你按下**`+` 号。
4. 让观众输入另一个数字（建议4-5位数）。
   * *观众输入：`8888`*
5. **你按下**`=` 号。
   * *屏幕显示：`10883`（正常结果）。*

### 第二阶段：下套（触发机关）

6. 看着这个结果，你对观众说：“这是一个普通的数字，但如果我们加上一点随机性呢？”
7. **关键动作**：你当着观众的面，**再次按下 `+` 号**。
   * *此时屏幕显示：`10883+`*
   * *（系统内心OS：机关已启动，目标锁定为当前时间！）*

### 第三阶段：奇迹（上帝模式）

8. 把手机/键盘推给观众：“现在，闭上眼，或者乱按，把屏幕填满。”
9. 观众开始疯狂按键（比如按 `7`、`9`、`C`、`⌫`）。
   * *屏幕实际显示：系统预设好的数字，比如 `2` `1` `6` `0`...*
10. 当观众发现按键不再出数字时（说明位数够了）。
11. 让观众按下 `=` 号。
    * *屏幕显示：`2171222`（假设现在是 2月17日 12:22）。*
12. 你拿出另一台手机/手表，展示当前时间：**2月17日 12:22**。
13. 全场掌声。

## ❓ 突发情况自救 (FAQ)

**Q: 观众手太快，数字没输完就按了等于号怎么办？**

* **A: 没事。** 程序有保护机制。如果数字没输完，按等于号**不会出结果**，只会像按了数字键一样多显示一位数。你只需淡定地说：“看来还没输完，继续按。”

**Q: 观众按了退格键（⌫）或清除键（C）怎么办？**

* **A: 没事。** 在这一步，这些功能键都被“黑”了。按退格键**不会删除**数字，反而会**增加**数字。这反而增强了魔术效果（“你想删？命运是删不掉的！”）。

**Q: 现在的日期是 10月、11月、12月 怎么办？**

* **A: 更安全。** 10月的时间数字是 8 位数（如 `10211230`，一千多万），你的安全范围更大了，前两步随便输都没事。
* **危险区是 1月**：1月的时间数字较小（如 `1150900`，一百万左右），此时前两步输入要稍微保守一点。
